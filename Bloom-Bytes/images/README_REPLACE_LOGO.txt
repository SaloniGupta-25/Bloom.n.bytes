To replace the site logo:

1. Save the provided Bloom & Bytes logo image as logo.png.
2. Place the file at images/logo.png (overwrite the existing file).
3. Clear your browser cache and reload the site to see the new logo.

Notes:
- The site references images/logo.png in index.html already.
- If you'd prefer a different filename, update index.html and style.css accordingly.
